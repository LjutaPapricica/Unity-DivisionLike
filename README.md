# Project Purpose
1. I like TPS(Third Person Shooter) games! So I want to make my own game like division
2. personal portfolio & toy project
3. to learn Unity & C#

# Key Features
- control player character
- player character animation
- control weapons
- spawn enemies
- HUD to display information
- support HDRP

# Third Party Plugins
- Outline Effect
- MiniMap
- EZ Camera Shake
